# Snake Game (Python)

A simple Snake game written in Python using the built-in `turtle` module.

## Requirements

- Python 3.9+
- Tk support enabled in your Python installation (required by `turtle`)

## Run

```bash
python3 snake_game.py
```

## Controls

- Arrow keys: move the snake
- `R`: restart the game

## Gameplay

- Eat red food to grow and increase your score.
- Avoid hitting the walls or your own body.
- The game gets faster as your score increases.
